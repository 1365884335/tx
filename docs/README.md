## docsify 知识库
### 需要环境
> 安装 [nodejs](https://nodejs.org/en/)

### 打开命令行
> win+R打开运行窗，输入 [cmd](https://baike.baidu.com/item/%E5%91%BD%E4%BB%A4%E6%8F%90%E7%A4%BA%E7%AC%A6?fromtitle=CMD&fromid=1193011) 后 回车 

### 安装docsify
>  npm install -g docsify-cli

### 查看docsify版本
> docsify -y

### 部署docsify
> - 创建文件夹 
> - cd 你创建的目录路径

### 初始化docsify
> docsify init

> 输入 y 确认


### 启用docsify
> docsify serve
